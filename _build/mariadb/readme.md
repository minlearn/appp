无



下载安装及用法📄
-----

1) 如果你是在devdeskos中执行  


```pveinst mariadb```  
跑完后bash /root/init.sh(如果有) 初始化服务器  
自动或手动透出端口服务



2) 如果你是在自己的vps中执行


```wget -qO- https://github.com/minlearn/appp/raw/master/_build/mariadb/mariadb_install.sh | bash```  

跑完后bash /root/init.sh(如果有) 初始化服务器  
自动或手动透出端口服务
