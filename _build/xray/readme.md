下载安装及用法📄
-----

1) 如果你是在devdeskos中执行   


```pveinst xray```  
跑完后bash /root/ip.sh 初始化ip, bash /root/token.sh初始化uuid，cat /root/client.txt查看客户端配置


2) 如果你是在自己的vps中执行


```wget -qO- https://github.com/minlearn/appp/raw/master/_build/xray/xray_install.sh | bash```  

跑完后bash /root/ip.sh 初始化ip, bash /root/token.sh初始化uuid，cat /root/client.txt查看客户端配置
